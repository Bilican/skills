# Productivity

General workflow tools, not code-specific.

## Model-invoked

The agent can fire these on its own when the task fits, or you can type them.

- **[audio-to-report](./audio-to-report/SKILL.md)** — Transcribe a local audio or video recording on-device with Whisper, then rewrite it as a themed, timestamp-free PDF report that loses no content.

## User-invoked

Reachable only when you type them (`disable-model-invocation: true`).

- **[learn](./learn/SKILL.md)** — Deeply understand a subject (a concept, document, or repo) by being taught and quizzed until you've demonstrated mastery, stage by stage.
